## caltraffic.com

California traffic camera viewer. Built with Astro. Deployed to Cloudflare Pages.

## Development

```bash
npm install
npm run dev        # Local dev server
npm run build      # Build to dist/
```

## Deploy

```bash
wrangler pages deploy dist/ --project-name=caltraffic
```
